# handPPT

手势控制PPT播放

手势识别参考 https://blog.csdn.net/weixin_45930948/article/details/115444916

## 环境

- Windows 10
- python 3.8.3
- mediapipe

## 功能

- 从当前页开始放映
- 点击
- 结束放映

## 使用

1. ```bash
    pip install mediapipe
    pip install cv2
    ```

2. ```bash
    python main.py
    ```

3. 打开PPT
